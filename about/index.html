About

WAAX

Web Audio API eXtension is a JavaScript framework that facilitates and supports the development of web-based music application. Its goal is to utilize the web browser as a music creation platform with the modern web technology such as Web Audio API and Web Components.

This project is a part of my doctoral research at CCRMA (The Center for Computer Research in Music and Acoustics) in Stanford University.
