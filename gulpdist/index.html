WAAX (Web Audio API eXtension) is a JavaScript library built on top of Web Audio/MIDI API to facilitate audio programming on Blink-based web browsers - Chrome, Chrome Canary and Chromium. The project started from a small wrapper for Web Audio API and now it has grown to a comprehensive framework for web music application including features such as Web Audio/MIDI abstraction, Polymer-powered modular GUI, Timebase(sequencer, tempo) and real-time collaboration support.

WAAX feature highlights:

  • Extending Web Audio API for ease of use
  • Web Audio Plug-in (WAPL) Interface
  • Parameter & Preset Management
  • Timebase, Transport and NoteList (a.k.a. Sequencer Engine)
  • Modular GUI elements powered by Polymer
  • Web MIDI API Integration (supported behind flag)
  • Complete Front-end Development setup: NPM, Bower, Grunt, Foundation, Polymer
  • Tested with Mocha and Chai: In-browser Test
