WX.Oscil

Description

Multi-waveform Oscillator.

Parameters

NameTypeDescriptionValues
pTypestringOscillator Waveform Type'sine', 'square', 'sawtooth', 'triangle', 'custom'
pFreqfloatOscillator Frequency
pGainfloatOscillator Gain
pDynamicbooleanDynamic Node'true', 'false'

Usage

var osc1 = WX.Oscil({ pType: 'sine', pFreq: 440, pGain: 0.25 });
osc1.stop(WX.now + 2.0);
WX.patch(osc1, WX.DAC);
