WX.ADSR

Description

Triggable envelope with multiple phases.

Parameters

NameTypeDescriptionDefault
pAttackfloatAttack time in seconds0.005
pDecayfloatDecay time in seconds0.015
pSustainfloatGain in sustain phase0.75
pReleasefloatRelease time in seconds0.05
pTimeConstantfloatTime constant for exponential release0.01 (-60dB)
pModestringVoice handling'polyphonic', 'monophonic'

Usage

var osc1 = WX.Oscil();
var env = WX.ADSR({ pAttack:2.0, pRelease: 2.0 });
WX.patch(osc1, env, WX.DAC);

env.noteOn(WX.now + 1.0);
env.noteOff(WX.now + 2.0);
